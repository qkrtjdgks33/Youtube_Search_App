# Youtube_Search_App

YouTube 검색 애플리케이션입니다.

## 프로젝트 구조

```
youtube-search-app/
├── src/          # React 소스 코드
├── server/       # 서버 코드
├── public/       # 정적 파일
└── package.json  # 프로젝트 의존성
```

## 시작하기

### 의존성 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm start
```


### 프로덕션 빌드
```bash
npm run build
```

## 기술 스택

- React
- TypeScript
- Node.js
- Express (서버)
